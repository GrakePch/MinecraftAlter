# MInecraftAlter

## Controls
| Key | Operation |
| --- | --- |
| A | Camera move to left |
| D | Camera move to right |
| W | Camera move up |
| S | Camera move down |
| Q | Sun orbit East to West |
| E | Sun orbit West to East |
| F1 | Default rendering |
| F2 | Show wireframes |
| F5 | Randomize terrain |
| F11 | Toggle full screen |
| Esc | Exit |
